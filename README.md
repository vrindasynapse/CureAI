# 💙 CureAI

An AI-powered Disease Prediction Web Application built using **Machine Learning**, **Flask**, **Python**, **HTML**, **CSS**, and **JavaScript**.

CureAI predicts possible diseases based on user-selected symptoms through a clean, modern, and interactive user interface.

---

## 🌐 Live Demo

🔗 https://cureai-330v.onrender.com/

---

## ✨ Features

- 🩺 Disease Prediction using Machine Learning
- 🔍 Live Symptom Search
- 🎯 Interactive Symptom Selection Chips
- 📌 Selected Symptoms Panel
- 🎨 Modern Responsive Dark UI
- ⚡ Fast Flask Backend
- 💻 Fully Responsive Design
- ☁️ Deployed on Render

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/vrindasynapse/CureAI.git
```

Move into the project

```bash
cd CureAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
